# Changelog

이 프로젝트의 주요 변경사항을 정리합니다. 최신 항목이 위에 오도록 날짜별로 기록합니다.

## 2026-07-24

### 고정 답변 규칙 (topicRules)
- 할인 답변에서 확정 수치("최대 10%") 제거 — 공식 가격 대비 더 저렴하게 제공한다는 안내로 변경
- 할인 답변 끝에 구체적인 할인율은 support 메일로 문의하라는 안내 추가 (수치를 묻는 질문에 답이 두루뭉술해지지 않도록)

## 2026-07-10 (계속)

### 다국어
- 위젯 UI 텍스트(헤더, placeholder, 전송 버튼, 인사말, 오류 메시지)가 브라우저 언어(`navigator.language`)에 따라 한/영으로 자동 전환 (기존에는 챗봇 답변만 언어에 맞춰 나가고, 위젯 자체 UI는 한국어로 고정되어 있었음)

### UI/브랜딩
- 채팅 패널 중앙에 AlRouter 로고 워터마크 추가 (alrouter.ai에서 로고 파일을 가져와 자체 서버에서 서빙)
- 첫 인사말에 "AI 챗봇이며 응대가 미흡할 수 있다"는 정체성 고지 문구 추가 (위젯 최초 인사말 + 서버 인사말 규칙 둘 다)

### 고정 답변 규칙 (topicRules)
- 할인율 규칙이 "얼마나 싸?/저렴해?" 같은 표현도 인식하도록 확장
- 지원 모델 문의 규칙 추가 — '모델 & 가격' 페이지 + support 메일 안내 (모델/가격은 수시로 변동되어 목록을 직접 답변하지 않음)

## 2026-07-10

### 안전/보안
- 욕설(금칙어) 필터링 추가 — 감지 시 모델 호출 없이 차단 + 담당자 알림
- 민감정보(주민등록번호/카드번호/전화번호) 자동 마스킹 — 저장·알림·모델 전달 전에 처리되어 원본이 남지 않음
- 브랜드명 표기(`AlRouter` / `alrouter` / `알라우터`) 정규화로 검색 결과 일관성 확보

### 신뢰성/안정성
- "답변 실패" 감지 정규식 보강 — 모델이 few-shot 예시를 따라 스스로 생성한 안내 문구도 인식하도록 개선
- 새로고침 시 대화 맥락이 끊기지 않도록 세션 종료 신호에 10초 유예 적용 (실제 탭 종료와 새로고침을 구분)
- 화면에 보이는 대화 내용이 새로고침/채팅창 열닫기에도 유지되도록 개선 (탭을 닫으면 초기화)
- 한글 입력기(IME) 조합 중 Enter 키로 인한 메시지 중복 전송 버그 수정

### 다국어
- 고객이 사용한 언어에 맞춰 답변하도록 변경 (한국어 질문엔 한국어, 영어 질문엔 영어)
- 대화 요약 알림도 한국어로만 작성되도록 프롬프트 보강

### 고정 답변 규칙 (topicRules)
- 인사말 규칙 추가 (한/영 자동 대응, 담당자 알림 없음), 이후 다양한 인사 표현으로 범위 확대
- 할인율, 계약 해지, 보안 취약점 신고, 결제 수단, 회원 탈퇴, 서비스 장애, 기업 문의 규칙 추가 (총 11개)

## 2026-07-07

### 배포
- 실 서버(Rocky Linux, GPU) 배포
- Cloudflare Tunnel로 `chatbot.alrouter.ai` HTTPS 도메인 연결
- 앱을 `127.0.0.1`에만 바인딩 (외부 직접 노출 차단)
- 위젯 임베드를 위한 CORS 허용 목록 추가
- 프론트엔드 핸드오프용 위젯 삽입 스니펫 문서화

### 모델
- `llama3.2` → `qwen2.5:14b` → `qwen3:32b` → `gemma4` 순으로 비교 테스트 후 `gemma4`로 최종 확정

### 지식 베이스
- Confluence 지식베이스를 AlRouter 관련 문서로만 한정 (다른 제품 문서 제외)
- 응답 내 원문 링크 자동 제거 (내부/제한 문서 유출 방지)

### 위젯
- 한글 입력기(IME) 조합 중 Enter로 인한 중복 전송 버그 최초 수정 (이후 07-10에 추가 보강)

### 기타
- 세션 메모리 누수 수정 (30분 유휴 세션 자동 정리, 대화 기록 20개로 캡)
- IP당 분당 15회 rate limit, 메시지 1000자 제한
- 임베딩(`bge-m3`) 기반 의미 검색 도입, temperature/컨텍스트 길이 튜닝, few-shot 예시 추가
